Change Sudoku Input and Highlight Settings

Open Preferences to change how numbers are entered and what the board highlights. Each setting is saved in your browser.

By Admin · Updated July 12, 2026

Choose how you enter numbers

Cell-first mode starts with a square. Digit-first mode keeps one number selected while you tap several squares.

Digit-first mode is useful when scanning the whole board for one number. Cell-first mode is useful when working through every candidate in one square.

Turn board highlights on or off

One setting shades the selected cell's row, column, and box. Another highlights matching digits and candidate notes.

Line counts are available from the board controls. They show how many times the selected digit can still appear in each row and column.

Choose whether placed digits clean up notes

Auto-remove notes deletes a placed digit from candidate notes in the same row, column, and box. You can turn it off when you want to manage every candidate yourself.
