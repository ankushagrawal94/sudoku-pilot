Skip the Boring Part of Sudoku Practice

Select the Sudoku techniques you already know and run them automatically. You can spend more of the puzzle looking for techniques you are still learning, without asking the app to reveal which one comes next.

By Admin · Updated July 12, 2026

Run the techniques you already know

Select the techniques you can already find reliably. Run selected techniques repeats those methods until none of them can move the puzzle forward.

For example, run Last Digit, Naked Single, and Hidden Single before you start looking for pairs or fish.

Leave the next technique unnamed

Select a set of familiar techniques instead of naming the harder technique you hope to find. The remaining board may contain a pair, fish, wing, or another supported pattern. You still have to decide what to look for.

This setup leaves the next technique unnamed. You still have to recognize the pattern on the board.

Ask the coach when you need it

Search the board for as long as the search feels productive. Open Hint when you want help. The first steps check the board and give a clue before the exact move is shown.

The full explanation marks the evidence, candidate eliminations, and placements. You can compare the proof with what you saw, then continue solving.

Use focused Practice mode when you want repetition

Practice selected technique serves a different kind of session. Choose one technique or a small group, and the app searches generated puzzles for a board where one of those methods is available.

Use that mode to repeat a pattern across several boards. Use Run selected techniques when you want a normal puzzle with the familiar work cleared away.

Set up a skip-the-boring-part session

  1. Open More, then open Techniques.
  2. Select the methods you can already recognize reliably.
  3. Open Run techniques and choose Run selected techniques.
  4. Study the remaining board without selecting the technique you expect to find.
  5. Open Hint when you want a clue or a complete explanation.
