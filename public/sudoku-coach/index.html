Get Help With the Next Sudoku Move

Open the coach when you get stuck. Sudoku Pilot finds the logical moves available on the current board and shows the evidence for each one.

By Admin · Updated July 18, 2026

See every move the coach found

A board often has more than one useful move. The coach lists all the moves it can prove with your enabled techniques.

Choose any result to see its technique and affected cells. The explanation uses different colors for evidence, removed candidates, and placed digits.

Ask for only as much help as you need

The hint starts by checking the board and pencil notes. It will point out a contradiction or incomplete notes before looking for a technique.

The four clues reveal the technique, one candidate, the search area, and the exact move. Press Apply when you want the deduction added to your board.

Limit hints to techniques you know

The Techniques panel controls what the coach may suggest. Turn off any method you have not learned yet.

For example, you can leave singles and X-Wing enabled while hiding Swordfish and the wing techniques. The coach will stay within that list.

Run the steps you no longer need to practice

Select one or more familiar techniques and run them automatically. Sudoku Pilot repeats them until none of the selected methods can move the board forward.

I use this to clear singles before practicing a harder pattern. You can review the board after each run.

Techniques the coach currently knows

Sudoku Pilot currently detects Last Digit, Naked Single, Hidden Single, Pointing Candidates, Claiming Candidates, Hidden Pair, Hidden Triple, Naked Pair, Naked Triple, Naked Quadruple, X-Wing, Swordfish, Skyscraper, 2-String Kite, XY-Wing, XYZ-Wing, and W-Wing.

The same list powers hints, practice mode, and automatic runs.
